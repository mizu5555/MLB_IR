## 🎯 核心內容速查

### 系統規格
- **數據量：** 6,133 筆記錄
- **賽季範圍：** 2022-2025（4 年）
- **統計項目：** 313 個
- **查詢類型：** 3 種（Factual, Ranking, Analysis）
- **準確率：** 100% (Query Classification)

### 核心文件
- `week2_query_classifier.py` - 查詢分類器
- `week2_smart_router.py` - 智能路由器
- `week2_mlb_assistant.py` - 主系統
- `week2_streamlit_demo.py` - Web UI

### 啟動命令
```bash
streamlit run week2_streamlit_demo.py
```

### 測試查詢
```
Factual: "Aaron Judge 2024 wRC+"
Ranking: "Who has the highest wRC+ in 2023?"
Analysis: "Why is Aaron Judge so good?"
```
